My Projects

AMAZON CLONE

The AMAZON Clone is a replica of the popular e-commerce platform Amazon, created using HTML, CSS, and JavaScript. The project features a fully responsive design with a modern user interface, displaying product listings, a shopping cart, and a simple checkout process. It uses JavaScript for dynamic interactions, such as adding items to the cart, calculating totals, and providing an interactive user experience. This clone mimics the basic structure and functionalities of Amazon, showcasing web development skills in building an e-commerce platform with core frontend technologies.

UNIVERSITY PORTAL

The University Portal is a web application designed to streamline communication and information access for students, faculty, and staff. Built with HTML, CSS, and JavaScript, this portal provides an intuitive user interface for managing academic resources, such as course schedules, grades, announcements, and event updates. The responsive design ensures accessibility across various devices, while JavaScript adds dynamic functionalities like interactive forms and real-time updates.

RESTRAUNT PORTAL

This website is designed for a restaurant, showcasing its menu, ambiance, and services. It features a clean and modern layout using HTML and CSS, with interactive elements like a contact form and navigation. JavaScript is used for dynamic actions such as form validation and smooth scrolling. The site is fully responsive, ensuring a seamless experience across devices. Users can easily navigate through sections like "About Us," "Menu," "Contact," and "Reservation," providing a user-friendly interface for both customers and restaurant staff.
